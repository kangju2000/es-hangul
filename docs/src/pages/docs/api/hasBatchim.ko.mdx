# hasBatchim

한글 문자열의 마지막 글자가 받침이 있는지 확인합니다.

````typescript
hasBatchim(
  str: string,
  options?: { only?: "single" | "double" }
): boolean


 ```typescript
 hasBatchim('값') // true
 hasBatchim('토') // false
 hasBatchim('갑', { only : "single" }) // true
 hasBatchim('값', { only : "single" }) // false
 hasBatchim('토', { only : "double" }) // false
````
